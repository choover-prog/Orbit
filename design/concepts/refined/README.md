# Refined Orbit Concepts

These three project-bound PNGs were generated independently with the built-in image generation workflow on July 16, 2026. They use fictional data and are visual exploration artifacts, not production UI.

## Quiet Orbit prompt summary

Create an almost-empty resting and attention experience on warm parchment. Show one fictional travel conflict as a large typographic moment, a thin functional relationship between the flight and meeting facts, one “Talk it through” action, a conversational input, and a quiet sentence acknowledging hidden secondary concerns. Use an open-circle Orbit mark, one blue action color, and at least 60% negative space. Avoid cards, lists, domain navigation, and decorative space imagery.

## Focus prompt summary

Create a single-task conversational workspace answering “What are my options?” Show three spacious text-row alternatives, one selected through a partial orbit stroke, one “Draft this” action, collapsed evidence, and a follow-up input. Do not use a dashboard, accumulated chat transcript, cards, timeline, or domain chrome.

## Ambient Orbit prompt summary

Create a restrained ambient environment where one relevant blue point arrives along a single cropped path and resolves into the fictional travel conflict. Keep one “Tell me more” action, conversational input, and a reassurance that nothing else needs attention. Use no other nodes, domain labels, cards, top navigation, or science-fiction ornament.
